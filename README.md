# MIYANOTE

## 概要

チーム学習用のTODO管理アプリケーション

## 機能

- 個人TODO管理
- プロジェクト・メンバー管理
- チーム学習ダッシュボード
- 議事録テンプレート機能

## 技術スタック

- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: Go + PostgreSQL
- Infrastructure: Docker
